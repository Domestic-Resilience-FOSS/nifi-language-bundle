Description:

This processor uses the Apache-licensed Lingua library to detect the language of a block of text extracted using RecordPath operations.

Configuration Properties:

  • Input RecordPath - The RecordPath operation that builds the block of text to use for analysis.
  • Output RecordPath - The RecordPath operation that finds the field where the language code will be inserted.
  • Unknown Language - Provide a default language code to use when the language cannot be determined.
